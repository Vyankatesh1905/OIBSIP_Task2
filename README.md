#  Personal Portfolio Website

A modern and responsive **Personal Portfolio Website** built using **HTML, CSS, and JavaScript**. This project highlights my profile, technical skills, education, projects, certifications, and contact details through a clean, interactive, and visually appealing interface.

##  Features

* Responsive design
* Dark & Light mode toggle
* Smooth scrolling navigation
* Animated sections
* Interactive project cards
* About Me section
* Skills showcase
* Education section
* Certifications gallery
* Contact information
* Mobile-friendly layout

##  Tech Stack

* HTML5
* CSS3
* JavaScript

## Getting Started

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser or run it using **Live Server**.
